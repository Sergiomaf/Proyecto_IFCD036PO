# Proyecto_IFCD036PO
## Introduccion
Este es el proyecto para el curso IFCD036PO de METODOLOGÍAS DE INGENIERÍAS DE SOFTWARE PARA LA WEB.
Se creara una aplicación web y se colocara todo el codigo fuente asi como la documentacion en un repositorio web.
## Entorno
### Hardware.
Es una aplicación web, por lo que se necesitara un servidor o alojamiento web capaz de contenerla y PCs capaces de navegar
por internet con las tecnologias que usemos. Si usamos Java deberá cumplir los requisitos de la versión que utilicemos, etc.
